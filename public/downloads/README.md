# Calcy Downloadable Tools

These assets are general planning templates for Australian mortgage research.

- Users should enter their own loan amount, interest rate, term, repayment frequency, and extra repayment assumptions.
- Sample rows are examples only and are not personalised lender quotes.
- The templates are general information only and are not financial advice.
- Future files should only be linked from the app after the real asset exists in this folder and tests confirm the URL is valid.
